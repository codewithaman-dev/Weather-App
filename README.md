## Weather App

A modern and responsive **Weather Application** that shows real-time weather information for any city.
Users can search for a city and instantly see the **temperature, humidity, and wind speed**.

---

## Live Demo

https://codewithaman-dev.github.io/Weather-App/

---

## Features

* Search weather by city name
* Displays real-time temperature
* Shows humidity level
* Displays wind speed
* Responsive and clean UI
* Fast API-based weather data

---

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Weather API
